### React + Vite

- [Vite url](https://vitejs.dev/guide/)

- npm create vite@latest
- npm install
- npm run dev
- if change the port number: got to package.json line number: 07
	```
	"scripts": {
		Old => "dev": "vite",   
		New => "dev": "vite --port=3000",
	}
	```
